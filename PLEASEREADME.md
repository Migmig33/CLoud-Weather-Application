
CLoud is an weather web application that is built in HTML, CSS, anud Javascript. 
it allows the ser to search any city and display real-time weather data from the WeatherAPI.
The application provides a clean and responsive, that display the current temperature, humidity, wind speed, and weather condition. 
It also include AI-like summary generator("CLoudBot) that interprets weather data and gives user a friendly, overview of current conditions.
User can also switch theme from dark mode to light mode, making the user experience visually adaptive and modern.

Key Feature:
City-Based weather search 
Current, 3-Day forecast, and 6-hour forecast
AI-like weather summary ("CLoudBot")
Responsive Design

Tech Stack Used:
HTML5
CSS
Javascript
WeatherAPI
